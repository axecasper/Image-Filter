## Image-Filter

### Abstract:
* Its a open source image filter code writen with python when I was second year student in university.

**Code's Functions:** 
* Gray scale Converter 
* Negative Converter
* Resizing the Image
* Rotating the Image
* Corping the Image
* Bluring the Image
* Sharpening the Image
* Smoothing the Image
* Changing Brightness of the Image
* Image Segmentation Applier
